# ApexForge Gym Platform

A modern responsive Gym and Fitness website built with HTML, CSS, Vanilla JavaScript, Node.js, Express, MongoDB, Mongoose, JWT, and bcrypt.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and update values if needed.
3. Start MongoDB locally.
4. Seed sample plans, trainers, supplements, and an admin user:
   ```bash
   npm run seed
   ```
5. Start the server:
   ```bash
   npm run dev
   ```
6. Open `http://localhost:5000`.
